# Security Policy

## Supported versions

Only the latest published release of Smooth Surfer is supported. Older versions
do not receive security fixes.

## Reporting a vulnerability

Please do not open a public issue for security reports.

Use GitHub's private vulnerability reporting instead: go to the
[Security tab](https://github.com/nishu-builder/smooth-surfer/security/advisories/new)
and click **Report a vulnerability**.

You can expect an initial response within 7 days. If a fix is warranted, it will
be released and the advisory published once users have had a chance to update.
